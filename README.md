# IFixTec - Sistema de Gestão de clientes

![InterfaceSistema](https://github.com/user-attachments/assets/e567592d-53b2-470a-b2e0-1a6af323a2b4)

## 📖 Sobre o projeto
Este projeto foi desenvolvido como uma aplicação desktop voltada para a gestão de serviços da empresa **IFixTec Tecnologia**.

O sistema tem como objetivo auxiliar no controle de atendimentos técnicos, clientes e registros de serviços, oferecendo uma interface simples e funcional.

Este é o meu primeiro projeto prático desenvolvido para um cliente real, marcando uma experiência importante no aprendizado e aplicação de conceitos de programação.

---

## 📋 Como baixar 

### 1. Requisitos necessários

[Java 8](https://www.java.com/pt-BR/download/)

[XAMPP(8.2.12, 8.1.25 e 8.0.30](https://www.apachefriends.org/blog/new_xampp_20231119.html)

[Workbench 8.0.46](https://dev.mysql.com/downloads/workbench/)

[Script SQL](https://github.com/diegoog26/IFixTec-Sistema/blob/main/assistenciatecnica.sql)

### 2. XAMPP

<img width="662" height="399" alt="image" src="https://github.com/user-attachments/assets/7ab934c2-a683-4b76-ba62-1000213e8371" />

#### Após instalar e abrir o XAMPP, aperte "Start" em "MySQL" e verifique a porta (3306, no caso da imagem). Não esqueça de ligar o XAMPP toda vez que for usar o programa ou configurar para ele funcionar automaticamente.

### 3. Workbench

<img width="1918" height="1031" alt="image" src="https://github.com/user-attachments/assets/d93ad0bc-f89e-4953-b4b7-4582de2317ae" />

#### Depois de ligar o XAMPP, clique no sinal de "+" para criar ma nova conexão SQL.

<img width="790" height="497" alt="image" src="https://github.com/user-attachments/assets/6127dc55-3930-4a52-9a33-37e841294997" />

#### Coloque um nome para a conexão e coloque a porta do XAMPP em "Port". O resto das opções, deixe exatamente como está na imagem e aperte "Ok".

<img width="1916" height="1029" alt="image" src="https://github.com/user-attachments/assets/01c88619-32bb-4fc4-92f9-c9968382273c" />

#### Com a conexão aberta, aperte em File(No canto superior esquerdo) e clique em "Open SQL Script". Selecione o Script SQL que está disponível nesse repositório. 

<img width="770" height="560" alt="image" src="https://github.com/user-attachments/assets/a03bcde1-f9c2-421a-9408-7a22fab77b93" />

#### Após isso, execute os três comandos(Selecione com o mouse em cima do comando e aperte Ctrl + Enter para executar). Não esqueça de executar de cima para baixo. 

#### Com o banco e a tabela criada, abra o executável JAR 

<img width="1008" height="530" alt="image" src="https://github.com/user-attachments/assets/10b483d1-3831-42ea-8208-3d73bac8c828" />

#### Verifique se o banco de dados está conectado. Se estiver, o programa está pronto para uso.
